# Little-Giant

## 项目概述

**游戏名称：** 进击的小巨人

**游戏类型：** 休闲类 3D 游戏

**运行环境：** Windows 7及以上、Linux、Android 4.0及以上、IOS、Windows 10 mobile

**游戏引擎：** Cocos2d-x 3.10



## Meet 进击的小巨人

- 鳥瞰視角
- 多人對抗
- 地圖限制



## 风格

- 休闲 + 竞技
- 策略 + 走位
- 多人 + 变数

Like a thrilling story book..
